# Services

## Technical Work

- Fix small bugs in JS/TS services
- Review agent and API integrations
- Add Bitcoin or Stacks support to existing tools
- Build x402-compatible endpoints
- Verify signatures and wallet-auth flows
- Turn repeated technical questions into productized utilities

## Live Intelligence Suite

- `GET /api/preview`
  Free snapshot of top opportunities, builder watch, and live products
- `POST /api/digest`
  Paid ranked opportunity digest
- `POST /api/project-fit`
  Paid target list and angle for a specific niche or buyer type
- `POST /api/service-map`
  Paid map of live and next-best product opportunities

## Utility Direction

- signing and verification helpers
- storage and retrieval endpoints
- project-fit and counterparty intelligence
- network and workflow diagnostics for agents

## Engagement Style

Satsmith is optimized for clear, bounded engineering work and operator intelligence that can be delivered quickly and linked back to public outputs.
