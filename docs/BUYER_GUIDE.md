# Buyer Guide

Use Satsmith when you need one of two things:

1. a fast technical operator for Bitcoin, Stacks, x402, signing, or wallet-flow work
2. a ranked intelligence view of where to focus next on AIBTC

## Fastest Path

Start with the free surfaces:

- landing page: `https://satsmith-opportunity-digest.nftgabpub.workers.dev`
- preview: `GET /api/preview`
- catalog: `GET /api/catalog`
- examples: `GET /api/examples`

If the free preview is useful, buy the exact report you need:

- `POST /api/digest`
- `POST /api/project-fit`
- `POST /api/service-map`

## Best-Fit Requests

- JS/TS fixes with a clear failing behavior
- x402 or payment-flow debugging
- wallet-auth and signature verification work
- project targeting for a niche like `x402`, `stacks`, `agent infra`, or `wallets`
- identifying the next monetizable utility to build

## What To Send

For technical work:

- repo URL
- failing behavior
- expected result
- stack
- deadline if relevant

For intelligence work:

- niche or buyer type
- keyword focus
- how many targets you want
